Projects

LED POV Arrangement using ARDUINO NANO

Team Size: 04 Role: Project Member

This project mainly emphasizes on the persistence of vision technology. Persistence of Vision (POV) occurs when a visual image seems to persist continuously when a stream of light is repeatedly interrupted for very brief instances and does not enter our eyes during those durations.

When a series of more images are presented in quick succession, it gives us an illusion that the images are in motion. This is the basic principle behind the making of the animated movies. A POV display has many advantages over a traditional CRT, LCD or LED display, like power savings, less complexity, easy configuration, attractiveness etc.

This board has a set of 8 LEDs that can be programmed to display the POV effect using the Arduino Nano and a dc motor. The display used is based on an Arduino Nano that is used to control the switching of 8 red coloured LEDs.
